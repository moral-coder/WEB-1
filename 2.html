Hypertext Markup Language (Html) is the standard Markup language for creating web pages and web applications. web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML descrives the structure of a aweb page seamantically and originally included cues for the appearnce of the document.

HTML elemets are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other times. HTML elemtes are delineated by tags, written using angle brackets.
